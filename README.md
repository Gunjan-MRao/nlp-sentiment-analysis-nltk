# nlp-sentiment-analysis-nltk
Title: Sentiment Analysis on Student Reviews (NLTK + pandas)

What this is
Classifies student written reviews into Positive/Neutral/Negative using NLP preprocessing and a weighted sentiment scoring approach.

Skills demonstrated
Python, pandas, NLTK, text preprocessing (tokenization, stopword removal, stemming), evaluation logic, reproducible analysis.

Method (high level)

Clean + preprocess text

Score using weighted positive/negative lexicon

Assign sentiment label and summarise trends

How to run

Create a Python environment

Install requirements

Run the notebook in notebooks/
